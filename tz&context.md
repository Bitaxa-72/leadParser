Цель проекта: создать базу данных в формате файла для ручной рассылки писем с продвижение своих услуг разработчик.

Задача: 
1. Парсинг данных в Нячанге касательно данного вопроса на основе открытых данных, требуются бизнесы с сайтом\без сайта с контактами почтой и соц сетью, с явно русской направленностью, т.е. бизнесы направленные на экспатов или сделанные экспатами. Требуется указывать что за тип бизнеса.
2. Создать скрипт для парсинга данных на определенное число записей, например 10 за раз или 100 за раз, ситуативно указывать будем.
3. Храним информацию в двух видах: 1. технический файл JSON, который будет отображать данные которые уже есть, чтобы не дублировать информацию каждый раз. 2. файл для меня в который попадают только уникальные записи, формат XLSX, он не перезаписывается, а каждый раз дополняется, каждая итерация парсинга записывает новую страницу так называемый лист, листы именуются по дате парсинга данных. 
4. Должна присутствовать возможность перенаправлять парсинг и на другие города мира. Под каждый город свои таблицы, хранящиеся в разных дирректориях.
5. Весь проект собрать в виде гит репозитория для публикации на гитхабе в будущем.
6. Проект собирает публичные данные, тисточники данных - карты, сайты офелированные с городом.
7. Нужные данные по столбцам: название бизнеса, тип бизнеса, город, сайт, почта, соц сеть (под каждую отдельный столбец), ссылка на источник данных в таблице.
